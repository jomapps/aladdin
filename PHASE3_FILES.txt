KEY FEATURES
============
✅ React Query setup with optimized defaults
✅ Complete query hooks for all entities (Projects, Episodes, Characters, Scenes)
✅ Orchestrator mutations (send message, create conversation, etc.)
✅ WebSocket provider with automatic reconnection
✅ Real-time event subscription and filtering
✅ Project context with computed values
✅ Optimistic updates with rollback
✅ Infinite scroll pagination
✅ Debounced values and callbacks
✅ Keyboard shortcut handling
✅ LocalStorage with cross-tab sync
✅ Global error boundary
✅ Error fallback components
✅ 404 and resource not found pages
